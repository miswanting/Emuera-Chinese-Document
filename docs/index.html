更完善

本文档不仅含有 Eramaker 与 Emuera 的全部官方文档的翻译，还有额外的指南、参考等原创内容。

更易用

参考现代脚本文档重新组织文档结构。并在日文文档的翻译部分也添加了大量示例代码。

更深入

从最简单的 Emuera 游戏开始，一直深入到对 Eramaker 和 Emuera 的源代码分析。

更通用

由于 Emuera 引擎本身设计定位存在特殊性，本文档在翻译与编写的时候进行了部分描述的脱敏。

更统一

原文档由于历史原因，组织得比较松散。本文档除翻译部分外，对源文档结构进行了全局重构与聚合。

更高效

本文档的部署使用了最新的文档框架 Vitepress。在表现力、编写和部署自动化都是及其便捷的。能够为文档内容提供一流的支持。
